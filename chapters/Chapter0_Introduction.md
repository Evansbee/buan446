## 1. WELCOME TO BUAN 446 (ONLINE)

Welcome to BUAN 446. This is an intensive, six-week graduate course delivered in a **fully online, asynchronous format**. This course will transform you from someone who may have never written a line of code into a data analyst who can wrangle messy datasets, automate repetitive tasks, and extract meaningful insights using Python.

**What This Course Is (and Isn't)**
This course is designed for practical, job-ready skills. We focus relentlessly on reading, cleaning, analyzing, and communicating data. This is NOT a computer science degree; it is a laboratory for data analysts.

**The Online Pace**
Because this course is online and spans only six weeks, the pace is rapid. You should expect to spend **12–15 hours per week** on videos, readings, DataCamp modules, and assignments. 

---
### ★ **THE 60-80% RULE**
You will spend 60-80% of your time as a data analyst cleaning and preparing data, not analyzing it. Real-world data is messy. This course embraces that reality.
---

## 2. THE CRAWL → WALK → RUN FRAMEWORK

We use a structured three-phase framework to teach you how to use AI (ChatGPT, Gemini, Claude) effectively without letting it become a crutch that prevents learning.

1.  **CRAWL Phase (No AI):** Building muscle memory. No AI allowed for readings, DataCamp, or Exams.
2.  **WALK Phase (AI as Tutor):** Use AI to explain confusing concepts or debug code you wrote. AI should help you *understand*, not do the work for you.
3.  **RUN Phase (Full Collaboration):** Substantial projects where AI collaboration is encouraged, provided you document its use and understand every line of code.

| **CRAWL** | **WALK** | **RUN** |
| :--- | :--- | :--- |
| Quizzes, Exams, DataCamp<br>**NO AI** | Concept explanation, Debugging<br>**AI FOR TUTORING** | Projects<br>**FULL AI COLLABORATION** |

## 3. THE CRESTVIEW STUDENT DATASET

We use a single comprehensive case study throughout the course: synthetic student enrollment data. You will move from a **Clean Dataset** (Weeks 1-2) to a **Messy Dataset** (Week 3) with 28+ intentional errors, and finally an **Extended Dataset** (Weeks 4-6) for multi-dimensional analysis.

## 4. COURSE POLICIES & DEADLINES

**The Sunday Deadline**
To maintain momentum in an online environment, the course follows a strict weekly rhythm:
> **All weekly deliverables—Assignments, Quizzes, and DataCamp modules—are due every Sunday by 11:59 PM Eastern Time.**

**Submission & Late Policy**
All work is submitted via Moodle. Late submissions are penalized **20% per day**. For Jupyter notebooks, ensure your code runs "Top to Bottom" without errors before submitting.

**Grading Breakdown**
* Weekly Quizzes (5 total): **15%**
* DataCamp Modules: **10%**
* Assignments (Weekly): **25%**
* Midterm Exam: **20%**
* Final Project: **20%**
